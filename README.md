# Mars Rover Challenge

#### Requirements

- Ruby 2.4.1
- Bundler
- RSpec 3.7


 #### Usage 
 
 - Clone the repo
 - CD into `pf45-mrc` 
 - Run the following to install th Rspec Gem if you want to run the tests.
 
 
      bundle install
      
 - To run the demo 
 
 
     ruby mars_rover_runner.rb
